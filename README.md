# WaveletTree
Wavelet-Tree project implemented by C#

## Get Start

1. Clone or download repo
2. Open `WaveletTree.sln` by Visual Studio 2019

## Development Environment

- IDE
    - Visual Studio 2019 Preview (16.0.0 Preview 4.0)
- .NET
    - .NET Standard 2.0
    - .NET Core 3.0
    - C# 8.0
    - MSTest
